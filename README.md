# English–Tamil Neural Machine Translation (Training)

This repository contains the training pipeline for an English–Tamil Neural Machine Translation model built from scratch using PyTorch.

## Overview

* Custom Encoder–Decoder Transformer
* Custom tokenizer and training pipeline
* No pretrained models used

## Hardware

Successfully trained on a 6GB VRAM GPU using memory-efficient training techniques.


## Inference & Deployment

Inference and deployment are available in a separate repository using FastAPI, Gradio, and Docker.

Inference Repository: [[link-to-your-inference-repo]](https://github.com/KAMALESH0081/en-ta-translation-service)

## Project Goals

* Understand Transformer architecture internals
* Build sequence-to-sequence models from scratch
* Learn efficient deep learning training workflows
